# Diseño

### Composición

Se propone una sola aplicación para profesores y estudiantes, en la cual al iniciar sesión se identificará el rol del usuario para que se redirija a la pantalla correspondiente. Se toma esta decisión puesto que permite la amplia reutilización de componentes, una característica valiosa dado que habrá funciones y datos a los que tanto profesores como estudiantes pueden acceder. Esto ofrece una experiencia más consistente, reduce el esfuerzo necesario para el mantenimiento y mejora la escalabilidad.
